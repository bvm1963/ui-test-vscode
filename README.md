## Prerequisites

• Node JS (v6.x.x or higher)

• NPM (v3.x.x or higher)

• Visual Studio Code (optional, for open project)

## Run the application

**Change the directory to your application’s root folder**

**Run the following commands in Command Prompt :**

cd ui-test-vscode

npm install

npm start
